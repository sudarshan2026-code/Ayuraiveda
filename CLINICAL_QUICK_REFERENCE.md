## 💊 Ama Detection Checklist

Track these 6 indicators:
- [ ] Irregular appetite (#22)
- [ ] Irregular digestion (#25)
- [ ] Slow digestion (#25)
- [ ] Constipation (#26)
- [ ] Frequent gas (#27)
- [ ] Low appetite (#22)

**Scoring:**
- 0 indicators = None
- 1-2 indicators = Mild
- 3-4 indicators = Moderate
- 5+ indicators = High

## ⚖️ Dosha Classification Rules

```
IF max_dosha > 50%:
    → Single Dominant (e.g., "Vata")

ELIF second_dosha >= 25%:
    → Dual Dosha (e.g., "Vata-Pitta")

ELSE:
    → Single Dominant (e.g., "Pitta")
```

## 📊 Sample Calculation

**Input:**
- Body frame: thin → V+3
- Skin type: dry → V+3
- Appetite: irregular → V+3
- Digestion: irregular → V+3
- Sleep: light → V+3
- Mental state: anxious → V+3
- (+ 53 more parameters)

**Calculation:**
```
Vata_score = 85 points
Pitta_score = 30 points
Kapha_score = 25 points
Total = 140 points

Vata% = (85/140) × 100 = 60%
Pitta% = (30/140) × 100 = 21%
Kapha% = (25/140) × 100 = 19%
```

**Output:**
- Prakriti: **Vata** (60% > 50%)
- Vikriti: **Vata aggravated**
- Agni: **Vishama Agni**
- Ama: **Moderate** (3 indicators)
- Risk: **High** (60% + 3 ama)

## 🌿 Recommendation Priority

### High Ama (5+ indicators):
1. **URGENT:** Panchakarma detoxification
2. Fast or light meals (kitchari)
3. Avoid heavy foods completely

### Moderate Ama (3-4 indicators):
1. Improve Agni with digestive spices
2. Eat only when hungry
3. Avoid overeating

### Mild Ama (1-2 indicators):
1. Drink warm ginger water
2. Regular meal times
3. Light dinner

### No Ama (0 indicators):
1. Maintain current diet
2. Preventive lifestyle
3. Seasonal adjustments

## 🎨 Dosha-Specific Recommendations

### Vata Imbalance:
- **Diet:** Warm, moist, cooked foods
- **Lifestyle:** Regular routine, oil massage
- **Avoid:** Cold, dry, raw foods
- **Herbs:** Ashwagandha, Bala
- **Yoga:** Gentle, grounding

### Pitta Imbalance:
- **Diet:** Cooling foods, sweet fruits
- **Lifestyle:** Stay cool, avoid heat
- **Avoid:** Spicy, sour, salty foods
- **Herbs:** Brahmi, Shatavari
- **Yoga:** Cooling pranayama

### Kapha Imbalance:
- **Diet:** Light, warm, spicy foods
- **Lifestyle:** Vigorous exercise, wake early
- **Avoid:** Heavy, oily, sweet foods
- **Herbs:** Trikatu, Guggulu
- **Yoga:** Energizing sequences

## 🎓 Quick Tips

### For Accurate Assessment:
1. Answer all 59 questions honestly
2. Consider current state (not ideal state)
3. Think about last 3-6 months
4. Don't overthink - first instinct is often correct

### For Interpretation:
1. Focus on dominant dosha first
2. Check Ama status - it affects everything
3. Read clinical justification carefully
4. Follow recommendations in priority order

### For Practitioners:
1. Use reasoning to explain to patients
2. Verify with pulse diagnosis
3. Consider seasonal variations
4. Track progress over time
