# 🚀 Deploy to Vercel via GitHub (No CLI Required!)

## This is the EASIEST way to deploy - No command line needed!

---

## Step 1: Create GitHub Account (Free)

1. Go to: **https://github.com/signup**
2. Enter your email address
3. Create a password
4. Choose a username
5. Verify your account
6. ✅ Done!

---

## Step 2: Create New Repository

1. Go to: **https://github.com/new**
2. Repository name: `ayurveda` (or any name you like)
3. Description: `AyurAI Veda - AI-Powered Ayurvedic Health System`
4. Select: **Public** (or Private if you prefer)
5. ✅ Click "Create repository"

---

## Step 3: Upload Your Files

### Option A: Via GitHub Website (Easiest!)

1. In your new repository, click **"uploading an existing file"**
2. Drag and drop these folders/files:
   - `api/` folder
   - `templates/` folder
   - `static/` folder
   - `vercel.json`
   - `requirements.txt`
   - `.vercelignore`
   - `README.md`

3. Add commit message: "Initial commit - AyurAI Veda"
4. Click **"Commit changes"**
5. ✅ Done!

### Option B: Via GitHub Desktop (Alternative)

1. Download GitHub Desktop: **https://desktop.github.com/**
2. Install and login
3. Clone your repository
4. Copy all project files to the cloned folder
5. Commit and push changes
6. ✅ Done!

---

## Step 4: Create Vercel Account (Free)

1. Go to: **https://vercel.com/signup**
2. Click **"Continue with GitHub"**
3. Authorize Vercel to access your GitHub
4. ✅ Done!

---

## Step 5: Import Project to Vercel

1. Go to: **https://vercel.com/new**
2. You'll see your GitHub repositories
3. Find your `ayurveda` repository
4. Click **"Import"**

---

## Step 6: Configure Project

### Framework Preset
- Select: **Other**

### Root Directory
- Leave as: `./`

### Build Settings
- Build Command: (leave empty)
- Output Directory: (leave empty)
- Install Command: (leave empty)

### Environment Variables (Optional)
- Click "Add" if you want to add:
  - `SECRET_KEY` = `your_secret_key_here`
  - `GROQ_API_KEY` = `your_groq_key_here`

---

## Step 7: Deploy!

1. Click **"Deploy"**
2. Wait 30-60 seconds
3. ✅ Your app is LIVE!

You'll get a URL like:
```
https://ayurveda-xyz123.vercel.app
```

## 🆘 Troubleshooting

### Issue: Files not uploading to GitHub
**Solution**: 
- Check file size (max 100MB per file)
- Upload folders one at a time
- Use GitHub Desktop for large uploads

### Issue: Vercel deployment fails
**Solution**:
- Check that `vercel.json` is uploaded
- Check that `api/index.py` exists
- Check that `requirements.txt` exists
- View deployment logs in Vercel dashboard

### Issue: 404 errors on pages
**Solution**:
- Check that `templates/` folder is uploaded
- Check that all HTML files are in `templates/`
- Check `vercel.json` routing configuration

### Issue: Static files not loading
**Solution**:
- Check that `static/` folder is uploaded
- Check that CSS/JS files are in correct folders
- Clear browser cache
