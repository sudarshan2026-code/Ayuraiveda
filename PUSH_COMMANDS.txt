# 🚀 EXACT COMMANDS TO PUSH TO GITHUB

## STEP 1: Create GitHub Repository First

Go to: https://github.com/new

- Repository name: ayurveda
- Description: AyurAI Veda - AI-Powered Ayurvedic Health System
- Public or Private: Your choice
- DO NOT check "Initialize with README"
- Click "Create repository"

Copy your repository URL (example: https://github.com/YOUR_USERNAME/ayurveda.git)

---

## STEP 2: Open Command Prompt

Press Windows Key + R
Type: cmd
Press Enter

---

## STEP 3: Navigate to Your Project

```bash
cd c:\Users\jayde\Documents\Ayurveda
```

---

## STEP 4: Configure Git (First Time Only)

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

Replace "Your Name" and "your.email@example.com" with your actual details.

---

## STEP 5: Run These Commands One by One

### Initialize Git
```bash
git init
```

### Add All Files
```bash
git add .
```

### Create First Commit
```bash
git commit -m "Initial commit - AyurAI Veda with all features"
```

### Add Remote Repository
```bash
git remote add origin https://github.com/YOUR_USERNAME/ayurveda.git
```

**IMPORTANT**: Replace `YOUR_USERNAME` with your actual GitHub username!

### Set Main Branch
```bash
git branch -M main
```

### Push to GitHub
```bash
git push -u origin main
```

---

## STEP 6: Enter Credentials

When prompted:
- **Username**: Your GitHub username
- **Password**: Use Personal Access Token (NOT your password)

### How to Get Personal Access Token:
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Note: "AyurAI Veda"
4. Select: ✅ repo (all)
5. Click "Generate token"
6. Copy the token
7. Paste it when asked for password

---

## ✅ DONE!

Your project is now on GitHub!

View it at: https://github.com/YOUR_USERNAME/ayurveda

---

## 🔄 TO UPDATE LATER (After Making Changes)

```bash
git add .
git commit -m "Description of your changes"
git push
```

---

## 🆘 IF YOU GET ERRORS

### Error: "git is not recognized"
**Solution**: Install Git from https://git-scm.com/download/win

### Error: "Authentication failed"
**Solution**: Use Personal Access Token instead of password

### Error: "Repository not found"
**Solution**: Check the repository URL is correct
