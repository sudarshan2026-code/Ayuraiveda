## 🆘 Common Issues & Solutions

### Issue: "Vercel CLI not found"
**Solution**: 
- Install Node.js first
- Then run: `npm install -g vercel`
- Or use GitHub method (no CLI needed!)

### Issue: "Don't want to use command line"
**Solution**: 
- Use GitHub method
- Follow `DEPLOY_VIA_GITHUB.md`
- No CLI required!

### Issue: "Deployment fails"
**Solution**:
- Check all files exist
- Verify `vercel.json` is correct
- Check deployment logs
- Try GitHub method

### Issue: "404 errors on pages"
**Solution**:
- Verify `templates/` folder uploaded
- Check file names match routes
- Clear browser cache
