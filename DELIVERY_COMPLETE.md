## 🎯 KEY FEATURES

### 1. Geometric Feature Extraction

```
┌─────────────────────────────────────────────────────────────┐
│                    5 KEY FEATURES                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1️⃣ Face Ratio (Width / Height)                            │
│     • Narrow (< 0.75) → Vata                               │
│     • Medium (0.75-0.9) → Pitta                            │
│     • Wide (> 0.9) → Kapha                                 │
│                                                             │
│  2️⃣ Jaw Ratio (Jaw Width / Forehead Width)                 │
│     • Pointed (< 0.8) → Vata                               │
│     • Angular (0.8-1.0) → Pitta                            │
│     • Wide (> 1.0) → Kapha                                 │
│                                                             │
│  3️⃣ Eye Size (Normalized to face width)                    │
│     • Small (< 0.10) → Vata                                │
│     • Medium (0.10-0.15) → Pitta                           │
│     • Large (> 0.15) → Kapha                               │
│                                                             │
│  4️⃣ Lip Thickness (Normalized to face width)               │
│     • Thin (< 0.05) → Vata                                 │
│     • Medium (0.05-0.08) → Pitta                           │
│     • Thick (> 0.08) → Kapha                               │
│                                                             │
│  5️⃣ Face Fullness (Area ratio)                             │
│     • Angular (< 0.60) → Vata                              │
│     • Balanced (0.60-0.75) → Pitta                         │
│     • Round (> 0.75) → Kapha                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🚀 QUICK START

### Installation (5 minutes)

```bash
# Option 1: Automated (Recommended)
setup_structural_analysis.bat

# Option 2: Manual
pip install mediapipe opencv-python numpy pillow
```

### Basic Usage (2 minutes)

```python
from structural_face_analysis import StructuralFaceAnalyzer

# Initialize
analyzer = StructuralFaceAnalyzer()

# Analyze
result = analyzer.analyze_face("image.jpg", input_type='path')

# Results
print(f"Dominant: {result['dominant']}")
print(f"Vata:  {result['scores']['vata']}%")
print(f"Pitta: {result['scores']['pitta']}%")
print(f"Kapha: {result['scores']['kapha']}%")
```
